##### 运行程序并读取输出
```
#!java
Process proc = Runtime.getRuntime().exec("hostname");
BufferedReader stdInput = new BufferedReader(new InputStreamReader(proc.getInputStream()));
String line = stdInput.readLine();
```

##### 写入文件
```
#!java

fileWriter.flush();
```
