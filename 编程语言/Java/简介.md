## 编译
```
#!bash
javac test/tools/DBTool.java
javac -cp .:lib/test.jar tools/DBTool.java
javac $(find app -name "*.java")
```

## 运行
```
#!bash
java -cp .:lib/test.jar tools.DBTool
```

## 状态
```
#!bash
#列举java进程
jps

#显示进程的内存状况
jmap -heap $Pid

##显示线程堆栈
jstack $Pid
```
