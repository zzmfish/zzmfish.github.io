##### jps
```
#!bash
#列举java进程
```

##### jmap
```bash
#显示进程的内存状况
jmap -heap $Pid

#显示内存对象直方图
jmap –histo:live $Pid
```

##### jstack
```bash
##显示线程堆栈
jstack $Pid
```
